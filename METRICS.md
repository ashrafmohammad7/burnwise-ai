# Metrics

## North Star Metric

The primary North Star metric for Burnwise AI is:

Completed audits per week.

This metric matters because it directly reflects:
- user intent
- engagement
- product usefulness
- top-of-funnel lead generation

The tool only creates value once a user completes the audit flow and sees optimization recommendations.

## Supporting Metrics

### 1. Audit Completion Rate

Percentage of users who start an audit and finish it.

This identifies friction in the workflow.

### 2. Email Capture Rate

Percentage of completed audits that convert into leads.

This measures business value for Credex.

### 3. Consultation Booking Rate

Percentage of high-savings users requesting a Credex consultation.

This directly measures commercial intent.

## Instrumentation

The first analytics events I would track:

- tool_selected
- audit_started
- audit_completed
- email_submitted
- consultation_clicked
- report_shared

## Pivot Signal

If fewer than 10% of completed audits lead to email capture after multiple iterations, I would reconsider:
- recommendation quality
- value perception
- target audience positioning
- onboarding flow
